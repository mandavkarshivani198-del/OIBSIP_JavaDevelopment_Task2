# OIBSIP_JavaDevelopment_Task2 Number Guessing Game

##Objective
Create a simple Number Guessing Game using Java.
The program generates a random number between 1 and 100, and the user must guess the correct number within a limited number of attempts.
This project helps beginners understand Java logic building, GUI development, event handling, and random number generation.

##Tools Used
•	Java
•	Java Swing
•	NetBeans IDE (Code Editor)
•	Java JDK

##Features
•	Start Game button to begin the game
•	Random number generation between 1 and 100
•	User input through text field
•	System hints such as Too High or Too Low
•	Maximum 10 attempts to guess the number
•	Score calculation based on attempts
•	Popup message when the user wins or loses
•	Reset button to start a new game
•	Clean and user-friendly GUI interface

##Steps Performed
1.	Created a Java project in NetBeans IDE.
2.	Designed the graphical interface using Java Swing components like JFrame, JLabel, JTextField, and JButton.
3.	Implemented random number generation using Java's Random class.
4.	Added game logic to compare user input with the generated number.
5.	Displayed hints such as Too High, Too Low, or Correct Guess.
6.	Limited the game to 10 attempts and calculated score based on attempts used.
7.	Added popup messages for winning and game over conditions.
8.	Tested the application to ensure proper functionality.

##Outcome
•	A fully functional Number Guessing Game application
•	Interactive graphical interface for user input
•	Logical implementation of guessing and scoring system
•	Demonstrates fundamental Java programming and GUI concepts
